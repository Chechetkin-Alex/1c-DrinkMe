# Бизнес требования

## Роли

### Гость

- просматривает каталог
- открывает карточку товара
- регистрируется или входит в аккаунт

### Пользователь

- добавляет товары в корзину
- редактирует количество товаров в корзине
- оформляет заказ
- смотрит историю заказов
- оставляет отзыв

### Администратор

- создает и редактирует категории
- создает и редактирует товары
- смотрит заказы
- меняет статус заказа

## Функциональные требования

### Каталог

- Система хранит категории товаров
- Товар относится к одной категории
- Товар имеет название, описание, цену, остаток, признак активности и тип
- Пользователь может получить список товаров
- Пользователь может фильтровать товары по категории и типу

### Корзина

- У авторизованного пользователя есть корзина
- Пользователь может добавить товар в корзину
- Пользователь может изменить количество товара
- Пользователь может удалить товар из корзины
- Система считает итоговую стоимость корзины

### Заказы

- Пользователь может создать заказ из корзины
- При создании заказа система копирует позиции корзины в позиции заказа
- После создания заказа корзина очищается
- Заказ имеет статус
- Пользователь видит только свои заказы
- Администратор видит все заказы

### Отзывы

- Авторизованный пользователь может оставить отзыв к товару
- У одного пользователя может быть один отзыв на один товар
- Пользователь может изменить или удалить свой отзыв
