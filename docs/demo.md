# Сценарий демонстрации

## Подготовка

Перед показом нужно выполнить:

```bash
cd backend
../.venv/bin/python manage.py migrate
../.venv/bin/python manage.py seed_demo
../.venv/bin/python manage.py runserver
```

В отдельном терминале:

```bash
cd frontend
npm install
npm run dev
```

Фронтенд открывается по адресу:

```text
http://127.0.0.1:5173/
```

Админка открывается по адресу:

```text
http://127.0.0.1:8000/admin/
```

Данные администратора:

- логин `admin`
- пароль `222333`

## Основной сценарий

1. Открыть главную страницу
2. Показать каталог товаров
3. Отфильтровать товары по категории
4. Открыть карточку Латте или Флэт уайт
5. Выбрать тип молока
6. Добавить товар в корзину
7. Перейти в корзину
8. Изменить количество товара
9. Оформить заказ
10. Открыть страницу заказов
11. Показать созданный заказ

## Сценарий авторизации

1. Открыть страницу входа
2. Создать пользователя
3. Для проверки студенческого признака указать почту с доменом `@phystech.edu`
4. После входа показать бейдж `МФТИ`
5. Показать, что корзина и заказы доступны только после входа

## Сценарий отзывов

1. Открыть карточку товара
2. Показать список отзывов
3. Добавить отзыв от авторизованного пользователя
4. Обновить список отзывов на странице

## Что показать в админке

- категории
- товары
- остатки товаров
- заказы
- статусы заказов
- отзывы
